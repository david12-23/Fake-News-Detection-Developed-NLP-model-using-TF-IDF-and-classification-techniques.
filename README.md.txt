# Fake News Detection System

Fake News Detection project developed using NLP techniques and machine learning algorithms.

## Features

- Detects fake news articles
- NLP preprocessing
- TF-IDF vectorization
- Flask web application
- Real-time prediction

## Technologies Used

- Python
- Flask
- Scikit-learn
- NLP
- TF-IDF
- Pandas
- NumPy

## Machine Learning

- TF-IDF Vectorizer
- PassiveAggressiveClassifier / Logistic Regression

## Run Application

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
